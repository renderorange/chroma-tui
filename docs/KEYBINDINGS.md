# Chroma Control Keybindings

## Global Keys

| Key | Action |
|-----|--------|
| `:` | Open command palette |
| `?` | Toggle help panel |
| `q` | Quit application |
| `ctrl+c` | Force quit |

## Effects List Mode

When the effects list is focused (left panel):

| Key | Action |
|-----|--------|
| `j` / `k` | Navigate up/down through effects |
| `enter` | Open parameters for selected effect |

## Parameters Mode

When the parameter list is focused (right panel):

| Key | Action |
|-----|--------|
| `j` / `k` | Navigate up/down through parameters |
| `h` / `l` | Decrease/increase parameter value |
| `enter` | Toggle parameter |
| `esc` | Return to effects list |

## Command Palette

| Command | Action |
|---------|--------|
| `quit` / `q` | Exit application |
| `help` / `h` / `?` | Open help panel |
| `settings` / `set` | Open settings screen |
